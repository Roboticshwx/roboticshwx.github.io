Supervised Learning

  • Regression: Learning input, output, or x to y mapping in order to predict numbers out.
  • Classification: Making a prediction of a category all of a small and discrete set of possible outputs.

This is different from regression which tries to predict any numbers all of the infinitely many number of possible outputs. There are only two possible outputs is what makes this classification.

Many more input values are required, in this case, such as the thickness of the tumor clump, uniformity of the cell size, uniformity of the cell shape and so on.

Feature Scaling

Feature scaling will enable the gradient descent to converge much faster.

Likewise, when the possible values of the feature are small, like the number of bedrooms, then a reasonable value for its parameters will be relatively large like 50.

A contour plot where the horizontal axis has a much narrower range than the vertical axis.

The key point is that the rescaling of the features can make the cost function look like a nice, symmetric bowl shape, which will make the gradient descent converge much faster.

To recap, when you have different features that take on very different ranges of values, it can cause gradient descent to run slowly but rescaling the different features so they all take on comparable range of values can speed up gradient descent significantly.

Normal method:

  • Feature scaling
  • Mean normalization
  • Z-score normalization
