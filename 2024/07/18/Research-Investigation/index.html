Dynamic Viewpoint-Based Obstacle Avoidance of Robotic Arms in Complex Environments

北京化工大学
2023
RA-L ISRIMT

这篇文章主要是研究一个移动机器人上的机械臂面对动态变化环境的路径规划，提出了一种局部环境构建策略来改进复杂环境中移动探测机器人机械臂的路径规划效率，通过构建局部环境进行路径规划，从而减少不必要的模型构建，提高避障效率。方法是 the axis-aligned boundary box (AABB) based on octree partitioning for collision detection analysis，通过仿真实验验证了这种方法的有效性。

Manipulator Reinforcement Learning with Mask Processing Based on Residual Network

西安电子科技大学
2023.5

该文章采用视觉支持的方法，使得UR机械臂能够在随机环境中学习抓取物体的技能，深度强化学习算法可以有效地学习视觉支持映射，以指导机械臂的抓取动作。为了验证设计的算法，本文在CoppeliaSim仿真平台上构建了一个机械臂抓取系统和测试场景。仿真结果表明，深度强化学习算法结合残差网络（ResNet）可以使机械臂获得较高的目标块抓取成功率。实验结果显示，该方法显著提高了机械臂的抓取效率。

Path Planning for Robotic Manipulator in Complex Multi-Obstacle Environment Based on Improved_RRT

2022.4
IEEE TMECH

本文提出了一种改进的快速随机树（Improved_RRT）方法，用于在复杂障碍环境中为机械臂规划成功且实际可执行的路径。

Improved_RRT方法通过混合约束采样、RRT_Connect与人工势场法结合、自适应引力场、动态步长和新节点约束策略的使用，有效提高了复杂障碍环境中机械臂路径规划的成功率和效率。仿真和实验结果表明，该方法在安全性、路径优化和综合性能方面具有显著优势。

  1. Introduction

介绍了一些传统的路径规划方法：

  • The particle swarm optimization.
  • genetic algorithm
  • APF method
  • A* algorithm

这些传统方法在高维路径中存在一些问题；

However, when applied to high-dimensional path planning problems, they are time-consuming and fall easily into the problem of local minimum.

然后是RRT算法，RRT算法因为大大降低了工作空间中障碍物建模不准确而导致的规划时间成本，因此广泛用于机械臂路径规划中；

但是RRT算法也具有收敛慢，容易陷入局部最优等问题；

后人改良的算法有：

  • RRT-Connect
  • RRT*
  • HD-RRT*
  • PGS_RRT*

改进后的Improved_RRT算法有三个创新点：

  • 采用混合约束采样代替原来的随机采样，得到的节点分布更接近障碍物之间的可通过区域，改进了RRT_Connect方法；

    Using hybrid constrained sampling instead of the original random sampling, the node distribution obtained by sampling is close to the passable area between obstacle;

  • 采用双竞争树和动态目标点，同时提出新的自适应步长和新的节点约束策略，减少了过度的探索和碰撞区域扩展；

    The method of double-competition tree and dynamic target points is used in the expansion. At the same time, an adaptive step size and new node constraint strategy are proposed.

  • 动态运动规划适用于不同的场景，并且优化了算法适用于各种实际场景；

  1. Collision Detection

采用 Flexible collision library (FCL) 进行碰撞检测，FCL是一种高效的碰撞检测库，可以检测机械臂的碰撞。

这个碰撞检测的实现是通过线，点和曲面的碰撞检测；

  1. Hybrid Constrained Sampling

在n维空间中采用高斯混合模型进行约束采样，得到的节点分布更接近障碍物之间的可通过区域，该文章在此基础上再进行了优化，得到了 hybrid constrained sampling.

  1. Adjacent Node Expansion

Standard RRT_Connect 算法的节点扩展速度非常慢

  1. Optimization & Experimental Verification

这一步的优化中解决了三个问题：

  • 路径规划中的局部极小值
    • 通过基于虚拟目标点的方法，使得机械臂逃脱局部极小值约束；
  • 在动态环境下的路径规划，常会遇到动态障碍物和路径中的奇异点，会对路径的稳定性和规划效率产生影响；
    • 改进动态路径规划算法，主要是局部重新规划和双树搜索策略，避免局部极小值点
  • 路径规划过程中可能会生成一些不必要的中间路径点，导致路径振荡，尤其是在起点和终点附近的障碍物区域。
    • 路径规划过程中可能会生成一些不必要的中间路径点，导致路径振荡，尤其是在起点和终点附近的障碍物区域。

Optimizing 7-DOF Robot Manipulator Path Using Deep Reinforcement Learning Techniques

IEEE International Symposium on Robotic and Sensors Environments (ROSE)

Machine Learning Department, MBZUAI, UAE

2024.7

探讨了三种不同的深度强化学习（DRL）方法：Deep Q-Network (DQN), Actor-Critic (AC), and Proximal Policy Optimization (PPO)，以优化7自由度（7-DOF）机器人机械臂（特别是Kinova Jaco辅助机器人）的路径规划。

通过仿真验证了这三种方法在静态环境中的路径规划能力。

结果显示，PPO在复杂环境中表现最为优越，能够更稳定地找到平滑的无障碍路径，而DQN和AC的表现稍逊一筹。
