  • Research on EVtol and UAV, trying to find some new ideas to improve the safety and stability of UAV;

I came up with an application for cooperative game theory in enhancing the safety of highly redundant systems like flying cars.

In a flying car equipped with 8 motors, each motor can be considered an independent decision-maker. Cooperative game theory can help these motors work together to ensure that even when some motors fail or cannot generate sufficient thrust, the flying car can maintain stability and safety, preventing system failure.

  • In flight mode, the 8 motors collaborate, using cooperative game theory to allocate thrust and ensure stable flight.

  • In the event of one or multiple motor failures or performance degradation, thrust is reallocated through cooperative game theory.

  • The cooperative game model should not only reallocate thrust but also automatically determine if a landing or mode switch is necessary. For example, in the case of multiple motor failures, the system could enter an emergency mode and execute a safe landing strategy.”

LLM + Robotics (Updated!)

In robotic experiments, it’s important to clearly distinguish between high-level and low-level actions. Low-level actions refer to specific steps like identifying, grasping, and moving lab equipment, such as a titrator. The goal is to use reinforcement learning (RL) to generate specific combinations of low-level actions for tasks in chemistry, biology, or materials science experiments.

An LLM can process a single high-level command (e.g., “add sodium carbonate solution to a test tube”) and translate it into a series of steps, such as:

Identify and grasp the titrator.
Extract the solution.
Align with the test tube.
Dispense the liquid precisely.
A predefined library of actions is used to execute these tasks, with RL fine-tuning the robot’s performance based on feedback. Sim-to-real learning helps with physical tasks, while real-world feedback, such as sensor data, helps set rewards for RL in more complex chemical tasks.

Conculsion:

  • Low-level actions: Grasping, moving, dispensing in lab tasks.

  • LLM role: Translates high-level commands into steps.

  • RL learning: Improves performance using feedback.
    Task library: Predefined actions for biological, chemical, and material experiments.
