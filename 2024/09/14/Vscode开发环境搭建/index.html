Vscode + ssh + X11 可视化开发

下载安装X11 server

VcXsrv Windows X Server
Xming X Server for Windows

安装完成之后打开XLaunch，选择Multiple windows，Start no client，Extra settings选择前两个选项，完成后保存配置文件。

设置系统变量

  • Windows客户端

    • 打开powershell, 输入setx DISPLAY "127.0.0.1:0.0", 然后重启终端;
    • 在终端输入echo $env:DISPLAY来验证, 输出为127.0.0.1:0.0即为成功.;

  • Linux服务器

    • 在~/.bashrc中添加export DISPLAY="localhost:10.0", 然后source ~/.bashrc;
    • 在终端输入echo $DISPLAY来验证, 输出为localhost:10.0即为成功.;
